# GistGift

GistGift is an AI-powered gift recommendation engine that analyzes photos, interests, personality cues, and social context to suggest meaningful, personalized gifts.

## Features
- AI gift recommendations
- Multi-photo analysis
- Saved gift lists
- Premium subscription tier
- RevenueCat integration
- iOS + Android support

## Tech Stack
- Lovable.dev
- Despia
- Supabase
- RevenueCat
- OpenAI API

## Purpose
This repo contains the codebase for the GistGift mobile application submitted to the App Store and Google Play Store.
